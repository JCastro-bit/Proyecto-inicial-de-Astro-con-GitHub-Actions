Este proyecto demuestra cómo usar GitHub Actions para compilar y desplegar automáticamente un sitio Astro a una rama separada.

Despliegue Continuo

Cada vez que hagas push a la rama principal, GitHub Actions compilará automáticamente tu proyecto y lo desplegará a la rama "prod". Esto facilita mantener tu sitio siempre actualizado.
